# :octocat: Hackathon.-Recognition-of-actions-based-on-skeletal-data
Команда № 5

## :star: Цель:
разработка прототипа системы распознавания действий человека (индивидуальных и групповых) по скелетным данным (ключевым точкам). Система должна работать на основе анализа поз человека в видео и классифицировать заданные типы действий.

## :pushpin: Задача от заказчика:
Необходимо изучить существующие методы распознавания действий по скелетным данным и разработать работающий прототип системы.

Решение должно быть обучено на открытых датасетах и поддерживать корректное распознавание следующих классов действий:

- Индивидуальные действия: сидение, ходьба, прыжки, курение.
- Групповые действия: драка, танец, рукопожатие, объятия, митинг, формирование круга/треугольника, перетягивание каната.
Система должна также уметь выделять групповые действия среди других активностей персонажей в кадре.

## :fire: Итог работы:
- Решение PoseLandmarker (mediapipe)
- Решение Ultralytics (Yolo8 Pose)
- Полноценный графический интерфейс GUI (с возможностью остановки кадра для обсуждения и рассмотрения его подробнее)
